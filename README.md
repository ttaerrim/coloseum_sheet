## `Installation`

```
# clone the project
$ git clone https://github.com/console-lo9/coloseum_sheet.git

# install modules
$ cd coloseum_sheet
$ npm install || yarn

# start
$ npm start || yarn start

⠀
⠀  You can now view this project in the browser.
⠀  http://localhost:3000/
⠀
```
